Undergraduate Information Systems Program

Connecting technology, business, and people.

Program Overview

UMBC’s IS undergraduate program equips students with both technical and organizational skills. Coursework covers programming, databases, cybersecurity, and systems analysis.

Sample Courses

  • IS 300 – Management Information Systems
  • IS 310 – Software and Hardware Concepts
  • IS 410 – Introduction to Database Design
  • IS 420 – Database Application Development

Career Paths

Graduates work as system analysts, cybersecurity specialists, data managers, and IT consultants.
